## Important keyboard shortcuts 
These shortcust work with **Android Studio**. They may require different combinations in other tools.

####Auto format code
	 CTRL + Alt + L 	(Windows/Linux)
	 CMD + Alt + L 	(Mac)
	 
####Quick comment
	 CTRL + / 	(Windows/Linux)
	 CMD + / 	(Mac)

####Refactor this
	 CTRL + T 	(Windows/Linux/Mac)
	
#####Quick Inspect
	ALt + Enter (Windows/Linux/Mac)
