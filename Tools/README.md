##Information
###Resizer_1.3.3.jar
*Credit: Redwarp 2014*

9Patch Resizer is a simple resizer tool for Android.
It takes a PNG image as input, and automatically renerates folders with several densities : ldpi, mdpi, hdpi, xhdpi, xxhdpi...
What's more : Resizer work on 9 patches as well ! Because, let's face it, it's a pain to have to do it by hand.
All icons were found on http://www.clker.com/
